# Basic Chatbot

A simple rule-based chatbot developed using Python as part of the CodeAlpha Python Programming Internship.

## Project Description

This chatbot interacts with users through the command line and provides predefined responses to specific inputs. It demonstrates the basic concepts of Python programming such as functions, loops, conditional statements, and user input handling.

## Features

- Responds to greetings like "hello"
- Answers basic questions
- Provides predefined responses
- Runs continuously until the user enters "bye"
- Simple and beginner-friendly implementation

## Technologies UsedS

- Python

## Concepts Used

- Functions
- While Loop
- If-Elif-Else Statements
- User Input and Output
- String Handling

## How to Run

1. Install Python on your system.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the following command:

```bash
python chatbot.py
```

##  Sample Interaction

```text
You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!

You: what is your name
Bot: My name is ChatBot.

You: bye
Bot: Goodbye!
```
